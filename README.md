# Literature Review Assistant

A minimalist, high-speed interface for literature reviews.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

## Features

- **Clean sidebar** with Gemini API key input and mode toggle
- **Two search modes**: Traditional Search and AI-Enhanced Discovery
- **Minimal UI** using `st.container` for focused workflows
- **No information overload** - only essential elements displayed

## Design Principles

Based on thematic analysis findings that researchers find current tools "clunky" and "bloated":

- Centered layout (max 900px) for focused reading
- Subtle visual hierarchy
- Progressive disclosure (expandable sections)
- Minimal color palette
- Fast, responsive interactions
